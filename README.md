# Portfolio
## Introduction
Ce portfolio a été créé avec les technologies suivantes :

- NuxtJS : Framework JavaScript pour créer des applications web modernes avec VueJS.
- Tailwind : Bibliothèque CSS utilitaire qui permet de créer des designs rapidement et facilement.
- I18N : Plugin NuxtJS pour la gestion de la traduction.
## Prérequis
Avant de commencer, vous devez avoir installé les éléments suivants :

- NodeJS : https://nodejs.org/
- Git : https://git-scm.com/
- Un éditeur de code (ex. Visual Studio Code, Sublime Text, Atom)

## Instalation
``` 
# clone
git clone https://github.com/Fullann/Portfolio.git

# install node == 14.0.1 and yarn (highly recommended)
$ npm install --global yarn

# cd into your project and install the dependencies
$ yarn install

# run the dev server
$ yarn dev

# open localhost:3000
```
